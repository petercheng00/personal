Personal Projects

A selection of random projects I've worked on in my own time.

SetCV
Card Game Frustration Reducer

Set is a card game where each individual card is easily recognized, but finding patterns among them is tricky. On the other hand, computers are great at finding patterns, but have trouble recognizing single objects

Currently a desktop application written in Python, this will eventually become an Android application as well as a more general visual card game framework

AutoPing
Network Latency Monitor

AutoPing is a pretty basic utility app. It simply repeatedly pings a server and lets you check if your internet is actually lagging or something else is going on - pretty handy for all the places I've lived where internet connection is terribly unreliable.

Berkeley Projects

Some of my favorite projects from my last 2 years at Berkeley.

Remoticon
CS294: Interactive Device Design

Remoticon is a fully-working hardware prototype coupled with an Android app. It allows the user to control their tv via smartphone, while storing favorites, setting reminders for showtimes, and receiving suggestions.

The case and internal PCB were entirely designed and printed by us, and the remote control IR frequencies were tuned to work on popular TV brands.

Hungarian Matching Visualizer
CS270: Combinatorial Algorithms and Data Structures

The Hungarian Matching Algorithm is a method of matching a bipartite set that results in optimal overall score across chosen connections.

Click here to try it out!

Handwriting Classification
CS289: Machine Learning

In this project, we attempted to correctly classify images of handwritten text as coming from either male or female writers. We generated 56 features for each document, by aggregating computations from sentences, words, and letters, and with an SVM with RBF kernel, achieved over 80% classification accuracy

Nostalgia
IEOR190E: Mobile Entrepreneurship

Nostalgia was a startup/app idea to reign in the older, tablet-using population, by reminding them of their youth and offering an attractive way to connect with their memories.

BTMesh
CS194: Mobile Device Development

This was my first serious foray into mobile development, and it was an ambitious one.

We created an app/framework with the ability to link together phones into a giant mesh network. The final prototype involved actual multi-node data transfer, a somewhat self-repairing mesh, and a pretty neat network visualizer

BOSS
CS169: Software Engineering
We collaborated with UCB's campus IT services to develop the Berkeley Online Student Scheduler. Not just a calendar system, it had to handle complicated rules, such as maintaining a certain level of different types of employees at each site and allowing for student employees to divide, move, and even trade work shifts among eachother.
